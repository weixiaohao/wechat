- 安装

  ```shell
  >>> git clone git@github.com:yhangf/wechat.git
  ```

  进入wechat目录

  ```shell
  >>> pipenv install
  ```

- 运行项目

  ```shell
  >>> pipenv shell
  >>> flask run
  ```
